Minecraft birds mod     
Available on [CurseForge](https://www.curseforge.com/minecraft/mc-mods/yarabirds/files) and [GitHub](https://github.com/Neisvestney/YaraBirds/releases)   
**Requires 
[Fabric Loader](https://fabricmc.net/use/installer/),
